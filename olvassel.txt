Backend Beállítása
.env fájl létrehozása
A /backend mappában található .env.example fájl nevét módosítsuk .env-re.
Alapesetben az SQLite adatbázist fogja használni az egyszerűsége és gyors tesztelhetősége miatt, 
de tetszés szerint módosítható a 22–27. sorban található beállításokkal.

Composer telepítése
Futassuk a következő parancsot a szükséges PHP csomagok telepítéséhez:

composer install

Alapértelmezett kulcs generálása
A következő parancs segítségével létrehozzuk az alkalmazás titkos kulcsát:

php artisan key:generate

Adatbázis migrációk létrehozása
A következő paranccsal létrehozzuk az adatbázis migrációkat:

php artisan migrate

A projektben a képek mappáját a következő paranccsal oszthatjuk meg:

php artisan storage:link 

Backend indítása
Végül a backend virtuális szerver elindításához használd a következő parancsot:

php artisan serve

Frontend Beállítása
Node függőségek telepítése
A /frontend mappában futtasd az alábbi parancsot a szükséges Node.js függőségek telepítéséhez:

npm install

Frontend szerver indítása
Az alábbi parancs segítségével indítsuk el a frontend szervert:

npm run dev

A link böngészőbe való másolásával elidul a projekt.

Az üres weblapon előbb regisztrálni kell. A weblap különbséget tesz felhasználó és Admin között.
Az ételek CRUD műveleteit csak Admin felhasználó tudja kezelni. Regiszráció(k) után az adatbázis users táblájában
az is_admin oszlopot ha TRUE értékre állítjuk, akkor a felhasználó Admin jogosultságot kap.
